## General Description:
The application allows students to take and organize notes by subjects, tags, and study groups.
Notes can include formatted text (Markdown), images, and attachments.

## Functional Requirements

Authentication using @stud.ase.ro account

Create, edit, and delete notes

Organize notes by subjects and tags

Attach files

Share notes

Create and manage study groups

## Non-Functional Requirements

Responsive interface (desktop/mobile)

Good performance (<500ms)

Secure data persistence

User interface in Romanian
